# Sublime插件
## 安装Package Control
[如何安装Package Control](https://packagecontrol.io/installation) （<font color=#0099ff>**推荐Simple方式**</font>）


## 安装插件
[如何安装插件](https://packagecontrol.io/docs/usage)

> `ctrl + shift + p` (Win, Linux) 
> `cmd  + shift + p` (OS X)

输入 --> `Package Control: Install Package` --> 要安装的插件名称
